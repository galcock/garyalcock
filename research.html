Summary

Density Field Dynamics (DFD) is a scalar-field framework in which a single field, psi, on flat three-dimensional space controls both how light propagates and how test masses accelerate.

  • Optics: the refractive index is n = exp(psi), so the local one-way speed of light is c multiplied by exp(−psi).
  • Dynamics: test masses feel an acceleration equal to (c squared divided by 2) times the spatial gradient of psi.

In the weak-field regime this reproduces standard post-Newtonian observables, while also predicting small, sector-dependent departures from Local Position Invariance that conventional metric gravity would set to zero.
